# WebRTC Screen Share App

## Overview

The **WebRTC Screen Share App** is a real-time screen sharing application that allows users to share their screens seamlessly with others using WebRTC technology. This app is built with Kotlin for Android, utilizing the WebRTC library for efficient video streaming and peer-to-peer connections.

**Note**: This application leverages **Wi-Fi Direct**, which allows for direct device-to-device connections without the need for a signaling server or TURN server. This enhances the app's performance and reduces latency during screen sharing.

## Technologies Used

- **Kotlin**: Main programming language for Android development.
- **WebRTC**: Open-source project that provides real-time communication capabilities.
- **Wi-Fi Direct**: Enables direct connections between devices.
